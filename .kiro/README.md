# TimeFlow Dashboard - Kiro Configuration

This directory contains Kiro-specific configuration and settings for the TimeFlow Dashboard project.

## Project Overview
A modern, professional time management dashboard that helps you track productive hours throughout your day.

## Features
- Real-time time tracking
- Commitment scheduling
- Productivity scoring
- Timeline visualization
- Dark/Light theme support
- LocalStorage persistence

## Tech Stack
- Pure HTML/CSS/JavaScript
- No dependencies
- Client-side only
- LocalStorage for data persistence

## Development
Simply open `index.html` in your browser to run the application.
